# 🎒 Inventory Test System

Простая демонстрация инвентарной системы с ограничением по весу. Проект создан для тестирования логики добавления предметов и контроля нагрузки.

## 🚀 Демо

Посмотреть работающую систему можно здесь:  
**[Inventory Test System](https://pavlovoiat.github.io/InventorySystem)**

## 🧩 Функционал

- Отображение текущего веса инвентаря (`текущий вес / лимит 5.0 кг`).
- Кнопки для добавления предметов с фиксированным весом:
  - 🍎 Яблоко (0.2 кг)
  - 📱 Телефон (0.3 кг)
  - 💧 Бутылка воды (0.75 кг)
- Контроль лимита: система не позволяет добавить предмет, если он превышает свободное место.
- В реальном времени обновляется общий вес инвентаря.

## 🌐 Технологии

- HTML
- CSS
- JavaScript (чистый, без фреймворков)

## 📌 Как это работает

1. Пользователь нажимает на кнопку предмета.
2. Скрипт проверяет: `текущий вес + вес предмета <= 5.0 кг`.
3. Если условие выполняется — вес добавляется, если нет — выводится предупреждение (через `alert` или консоль).
4. Отображение веса обновляется автоматически.
